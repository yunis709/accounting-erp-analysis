# 3. وحدات النظام (System Modules)

| الوحدة           | أهم الوظائف                 | ترابطها مع باقي النظام             |
|------------------|----------------------------|-------------------------------------|
| نظام المحاسبة    | شجرة حسابات، قيود يومية، ترحيل، تقارير مالية | الأساس لجميع الحركات المالية         |
| نظام المبيعات    | فواتير بيع، نقاط بيع POS، مرتجعات، خصومات | تغذية الحسابات والمخزون              |
| نظام المشتريات   | طلبات شراء، فواتير شراء، مرتجعات للموردين | مرتبطة بالمخزون، الحسابات، تقارير الشراء|
| نظام المخزون     | منتجات، وحدات، فئات، جرد، باركود، إدارة المستودعات | نقطة الالتقاء بين المبيعات والمشتريات  |
| نظام العملاء     | بيانات العملاء، كشف حساب وأرصدة، إدارة الديون | يؤثر على المبيعات والمالية            |
| نظام الموردين    | بيانات الموردين، أرصدة، كشف حساب          | يؤثر على المشتريات والمالية            |
| نظام التقارير    | تقارير مالية، أداء، أرباح، مخزون          | يعتمد على جميع وحدات النظام           |
| نظام المستخدمين  | إدارة المستخدمين والأدوار والصلاحيات       | حماية النظام وتسهيل الإدارة           |
| نظام الفروع      | إدارة الفروع والمخازن واستقلال الحسابات    | يتيح التوسع لعدة فروع                 |
| نظام العروض      | إدارة عروض الأسعار والتخفيضات             | داعم للمبيعات وتسويق المنتجات          |
| نظام التنبيهات/SMS| إشعارات دين، تقارير آلية للمالك            | يعزز المتابعة والرقابة                 |

**مخطط ترابط وحدات النظام نصيًا:**
```
[محاسبة]
  |    \_ [مبيعات] -- [مخزون] -- [مشتريات]
  |     |    |         |           |
  |   [عملاء] [منتجات] [موردين]    |
  \_______[تقارير] _____|_________/  
        |         \___ [تنبيهات/SMS] 
        \___ [مستخدمين وصلاحيات]
        \___ [فروع ومخازن]
``
