# 5. نظام المبيعات (Sales System)

## السمات الرئيسية
- تسجيل فواتير بسلاسة (كاش، آجل، POS)
- دعم الخصومات، الضرائب، العروض حسب الشروط
- نقاط بيع متعددة: أجهزة، شاشات، واجهات
- دعم طابعات حرارية وباركود للعمل السريع
- معالجة المرتجعات بسهولة ومتابعة أرصدة العملاء

## العمليات الأساسية
| العملية    | الخطوات الرئيسية           | التكامل مع النظام      |
|------------|---------------------------|-------------------------|
| فاتورة بيع | اختيار عميل/منتج/كمية – الدفع/الأجل| محاسبة – مخزون – حسابات عملاء|
| مرتجع      | ربط بفاتورة – خصم من المخزون/دين| محاسبة – مخزون – حسابات|
| خصومات     | لكل منتج أو عامة           | تقارير تفصيلية         |
| عروض      | حملات على منتجات مواسم      | تقارير وDashboards      |

**مخطط نصي للمبيعات:**
```
[واجهة البيع] -> [معالجة الخصومات/العروض]
    |                  |
[إصدار فاتورة] -- [توليد قيد محاسبي]
    |                  |
[تحديث مخزون] -- [تسوية مع عميل]
```
