# Technology Stack


## Overview
This document outlines the technology stack used in the ERP accounting system, highlighting the tools and frameworks that enable development and deployment.

## Frontend
- **Framework**: React.js for building user interfaces.
- **Styling**: Tailwind CSS for designing responsive layouts.

## Backend
- **Language**: Node.js for server-side logic.
- **Framework**: Express.js for handling requests and routes.
- **Database**: MongoDB for flexible data storage.

## Deployment
- **Platform**: AWS for hosting and managing the application.
- **CI/CD**: GitHub Actions for automated testing and deployment.

---

## Conclusion
The chosen technology stack provides robustness, scalability, and flexibility to the ERP accounting system.
