# 4. نظام المحاسبة (Accounting System)

## 1. شجرة الحسابات شاملة ومرنة
- دعم أنواع الحسابات: أصول، التزامات، إيرادات، مصروفات، رأس مال
- ترقيم آلي ويدوي، مستويات لا نهائية

**مثال شجرة حسابات:**
```
1- الأصول
   1.1- النقدية
   1.2- المخزون
   1.3- العملاء
2- الالتزامات
   2.1- الموردين
   2.2- الديون قصيرة الأجل
3- رأس المال
...
```

## 2. القيود اليومية (Journal Entries)
| رقم القيد | التاريخ   | الحسابات المدينة | الحسابات الدائنة | المبلغ | البيان | المرفقات |
|-----------|-----------|------------------|------------------|--------|--------|----------|
| 1         | 2026/04/01| الصندوق          | المبيعات         | 1000   | بيع نقدي| صورة      |
| 2         | 2026/04/01| العملاء          | المبيعات         | 500    | بيع آجل | صورة      |

## 3. دفتر الأستاذ
- عرض حركات كل حساب بتسلسل زمني

## 4. ميزان المراجعة والميزانية
- ميزان مراجعة يومي/شهري ومقارنات سنة/سنة
- ميزانية عمومية (Balance Sheet)
- حساب الأرباح والخسائر

## 5. القيود الآلية (Auto-Posting)
- تسجيل محاسبي تلقائي عند أي عملية بيع/شراء/مرتجعة/سداد

## 6. المصروفات/الإيرادات
- إدارة مصروفات التشغيل، الفواتير الدورية، تتبع الإيرادات حسب البنود
