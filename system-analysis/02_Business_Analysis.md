# 2. التحليل التجاري (Business Analysis)

## 1. أهداف العميل الأساسية
- إدارة جميع العمليات المالية والمخزنية للمحال/السوبرماركت بسهولة ودقة
- مراقبة العمليات البيعية (كاش، أجل، POS) وفصل صلاحيات الكاشير والمديرين والمحاسبين
- ربط الفروع والمخازن في منصة واحدة والتحكم في المخزون والأرصدة بدقة
- متابعة العملاء والديون والتسوية المالية بسهولة وسرعة وموثوقية
- دعم تقنيات حديثة (SMS، Dashboard ذكية، تقارير حية وتوقعات مخزون)

## 2. احتياجات العمل الرئيسية والألم
| الاحتياج/المشكلة            | شرح                             | الحل بالنظام المقترح                 |
|-----------------------------|---------------------------------|--------------------------------------|
| ازدواجية البيانات           | تكرار إدخال بيانات في أكثر من مكان| قاعدة بيانات موحدة – منع التكرار     |
| فقدان السيطرة على الديون    | صعوبة متابعة أرصدة العملاء      | كشف حساب آني + SMS تنبيهات           |
| بطء تسجيل العمليات          | سلاسل إدخال تقليدية مملة        | واجهة سريعة، باركود، عمليات آلية      |
| صعوبة الجرد وتتبع الأصناف   | فقدان مخزون أو أصناف            | جرد آلي + منظمة وحدود إعادة الطلب     |
| غياب التقارير التحليلية     | تحليل الأداء والمنتجات ضعيف      | Dashboards + تقارير ذكية             |
| عدم تكامل الفروع والمخازن   | تعدد قواعد البيانات             | منصة موحدة، إدارة صلاحيات متقدمة      |

## 3. مستفيدي النظام (Personas)
| الأداء                     | المهام الرئيسية                                      | احتياجاته من النظام                              |
|----------------------------|----------------------------------------------------|--------------------------------------------------|
| مالك المتجر                | الرقابة، التقارير، العروض                           | تقارير يومية، Dashboard ذكية، SMS فوري           |
| المدير المالي              | ضبط الحسابات، مراقبة الديون والتدفقات النقدية       | قيود آلية، تقارير تحليلية، كشف حساب دقيق         |
| الكاشير                    | تسجيل فواتير، عمليات الدفع السريعة                  | واجهة بسيطة وسريعة، دعم الطابعات والباركود        |
| أمين المخزن                | الجرد، مراقبة الحدود، تسجيل الحركة                   | شاشات جرد، تنبيهات نفاد أصناف، باركود            |
| مدير الفروع                | إدارة مستخدمين، مراجعة أداء الفروع                 | إدارة صلاحيات، تقارير الفروع، مراقبة حركات البيع  |

## 4. السيناريوهات التجارية الأساسية
- إدخال فواتير مبيعات يومية متنوعة (كاش/آجل/خصومات/مرتجعات)
- دخول بضائع جديدة إلى المخزن مع إشعار بالمخزون وحدود إعادة الطلب
- متابعة العملاء والمدفوعات الآجلة وتنبيههم عند حلول الاستحقاق
- إصدار تقارير أرباح يومية/شهرية/حسب الأصناف والأقسام
- إجراء تسويات ومقارنات مخزون دوري مع متابعة أي نقص أو زيادة
